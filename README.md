⚙️ Installation

# Clone the repo
git clone https://github.com/your-username/clothing-store.git

# Navigate
cd clothing-store

# Create virtual environment
python -m venv venv

# Activate
venv\Scripts\activate   # Windows
source venv/bin/activate  # Linux/Mac

# Install dependencies
pip install -r requirements.txt

# Run project
python run.py

---

🧪 Seed Data

python seed.py
