⚙️ Installation
# Clone the repo
git clone https://github.com/iblamesrix/LitWear.git

# Navigate
cd LitWear

# Create virtual environment
python -m venv venv

# Activate
venv\Scripts\activate   # Windows
source venv/bin/activate  # Linux/Mac

# Install dependencies
pip install -r requirements.txt

# Run project
python run.py


🧪 Seed Data

python seed.py
